# Power_BI_Dashboards

## Overview
This repository contains additional Power BI dashboards developed during my coursework.
The projects focus on data analysis, visualization, and KPI tracking using real-world datasets.

---

## Objective
### The goal of these dashboards is to practice:
- Building interactive reports in Power BI;
- Defining and tracking key business metrics (KPIs);
- Performing basic data analysis and segmentation;
- Visualizing insights for business decision-making.

---

## Tools Used
- Power BI
- DAX (basic measures)
- Data modeling
- CSV datasets
